## Tuple

- 순서있음
- 중복 가능
- 수정 불가능
- 삭제 불가능

```py
# 선언
a = ()
b = (1,)
c = (1,2,3,4)
d = (1,10,100,('a','b','c'),1000)

print(a) # ()
print(d[2:]) # (('a','b','c'),1000)
print(c+d) # (1,2,3,4,1,10,100,('a','b','c'),1000)
print(c*2) # (1,2,3,4,1,2,3,4)

# 함수
print(3 in z) # True
print(c.index(3)) # 2
print(c.count(4)) # 1
```
